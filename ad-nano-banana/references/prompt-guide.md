## 结构模板

**推荐结构:**
```
[主体] + [动作/状态] + [环境/背景] + [氛围/情感] + [艺术风格] + [技术细节]
```

**示例:**
```
一只橘色的猫咪坐在窗边,窗外是美丽的花园,温馨宁静的午后,水彩画风格,柔和的光线,高细节
```

**简化版（快速使用）:**
```
[主体] + [关键特征] + [风格]
```

**长度建议:**

| 长度 | 适用场景 |
|-----|---------|
| 10-20 词 | 简单主体 |
| 20-50 词 | 日常使用（推荐）|
| 50-100 词 | 复杂场景 |

## 风格指定

**绘画风格:** 水彩画、油画、素描、线稿、国画、浮世绘、波普艺术、抽象艺术

**摄影风格:** 肖像摄影、风景摄影、电影感、黑白、微距、长曝光

**数字艺术:** 赛博朋克、蒸汽朋克、低多边形、像素艺术、3D渲染、概念艺术

## 细节控制

**光照关键词:** 柔和的光线、戏剧性光线、金色时刻、背光、侧光、霓虹灯光

**色彩关键词:** 温暖色调、冷色调、高饱和度、莫兰迪色、单色

**构图关键词:** 居中构图、三分法、对称、特写、全景、鸟瞰视角、低角度

**细节程度:** 高细节、精致的、简约的、清晰的、柔焦的

## 常见场景模板

### 人物肖像
```
[人物特征] + [表情/动作] + [服装] + [背景] + [光线] + [风格]
```
示例: `年轻女性,自信的微笑,职业装,简约背景,柔和前侧光,专业摄影风格`

### 风景摄影
```
[地点/地形] + [天气/时间] + [光线] + [氛围] + [风格]
```
示例: `雪山湖泊,日出时分,金色光线,宁静祥和,风景摄影,高动态范围`

### 抽象艺术
```
[形状/图案] + [色彩] + [纹理] + [风格] + [情感]
```
示例: `流动的线条和色块,渐变蓝紫色,柔滑纹理,抽象表现主义,宁静神秘`

### 概念艺术
```
[主题] + [环境] + [氛围] + [风格] + [技术细节]
```
示例: `未来城市,垂直农场,赛博朋克风格,高细节3D渲染,电影级质量`

## 图生图 / 多图融合提示词

**风格转换模板:**
```
转为[目标风格],保持[要保留的元素],[额外要求]
```
示例: `转为油画风格,保持主体和构图不变,增加笔触质感`

**图像优化模板:**
```
提升[质量方面],增强[细节方面],[保持原有特征]
```
示例: `提升清晰度和细节,增强色彩饱和度,保持自然真实感`

**多图融合模板:**
```
融合这些图像的[特定方面],创造[期望效果],[技术要求]
```
示例: `融合这些图像的色彩和构图,创造独特的视觉冲击,保持和谐统一`

## 常见错误

| 错误 | 示例 | 改进 |
|-----|------|------|
| 过于简短 | `猫` | `一只橘色的猫咪坐在窗边,水彩画风格` |
| 自相矛盾 | `白色的黑猫` | `黑色的猫咪` |
| 关键词堆砌 | `猫,窗户,阳光,4K,8K,超清晰,完美...` | 保留核心描述即可 |
| 模糊不清 | `好看的风景` | `雪山湖泊,日出时分,金色光线` |
| 忽略风格 | `一只猫坐在窗边` | `一只猫坐在窗边,水彩画风格` |
