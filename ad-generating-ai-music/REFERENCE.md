# 参数与输出参考

## 命令行参数

### 模式选择（互斥）

| 参数 | 说明 |
|------|------|
| （默认） | 简化模式，只需 `--prompt` |
| `--custom` | 自定义模式，需要 `--prompt`/`--style`/`--title` |
| `--query` | 查询任务状态，需要 `--task-id` |

### 简化模式参数

| 参数 | 必需 | 说明 |
|------|------|------|
| `--prompt` | 是 | 音乐描述（如需指定男女声，在描述中说明，如"欢快的女声流行歌"） |
| `--instrumental` | 否 | 生成纯音乐（无歌词） |

注意：简化模式不支持 `--vocal-gender` 参数，通过 prompt 描述来控制人声性别。

### 自定义模式参数

| 参数 | 必需 | 说明 |
|------|------|------|
| `--prompt` | 有歌词时必需 | 歌词内容 |
| `--style` | 是 | 音乐风格，如 "流行, 抒情" |
| `--title` | 是 | 音乐标题 |
| `--instrumental` | 否 | 纯音乐（使用时不需要 --prompt） |
| `--vocal-gender` | 否 | 人声性别：`male` 或 `female` |

### 轮询控制

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--no-wait` | false | 提交后立即返回 |
| `--max-wait` | 300 | 最大等待秒数 |
| `--poll-interval` | 10 | 轮询间隔秒数 |

### 输出控制

| 参数 | 说明 |
|------|------|
| `--json` | JSON 格式输出到 stdout |

## 输出格式

### 文本输出（默认）

```
============================================================
音乐 #1
============================================================
音乐标题: music_1
下载链接: https://xxx.xxx/audio/xxx.mp3
```

### JSON 输出（--json）

```json
[
  {
    "title": "music_1",
    "audioUrl": "https://xxx.xxx/audio/xxx.mp3"
  }
]
```

## 任务状态

| 状态 | 说明 |
|------|------|
| `pending` | 等待处理 |
| `processing` | 生成中 |
| `completed` | 完成 |
| `failed` | 失败（查看 err_msg） |
