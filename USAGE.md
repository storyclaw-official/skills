# giggle-skills 使用指南

本指南面向普通用户，介绍如何安装和使用 giggle-* 系列技能。

---

## 技能列表（giggle-* 系列）

| 技能 | 功能 | 是否需要 API Key |
|------|------|----------------|
| `giggle-drama` | AI 短视频生成（多种画风） | 是 |
| `giggle-aimv` | AI 音乐视频 / MV 生成 | 是 |
| `giggle-music` | AI 音乐生成 | 是 |
| `giggle-image` | AI 图像生成（Seedream 模型） | 是 |
| `giggle-screenplay` | AI 剧本创作（姜文风格等多种风格） | 否 |

所有 giggle-* 技能共用同一个 `GIGGLE_API_KEY`，只需配置一次。

---

## 第一步：安装技能

将技能目录复制到 OpenClaw 的技能目录 `~/.openclaw/skills/`。

```bash
# 进入技能仓库目录（替换为实际路径）
cd /path/to/storyclaw-skills

# 按需复制，可选择安装部分或全部技能
cp -r giggle-drama ~/.openclaw/skills/
cp -r giggle-aimv ~/.openclaw/skills/
cp -r giggle-music ~/.openclaw/skills/
cp -r giggle-image ~/.openclaw/skills/
cp -r giggle-screenplay ~/.openclaw/skills/   # 无需 API Key
```

安装依赖（仅需执行一次）：

```bash
pip install requests python-dotenv
```

---

## 第二步：获取 API Key

`giggle-screenplay` 无需 API Key，可跳过本步骤，直接阅读下方各技能使用说明。

其余 giggle-* 技能均需要 **Giggle.pro API Key**，获取步骤：

1. 登录 [giggle.pro](https://giggle.pro)
2. 点击右上角头像，进入「个人设置」
3. 找到「API 管理」或「API Key」页面
4. 点击「创建 API Key」，复制并妥善保存

> **注意**：API Key 只显示一次，请立即保存到安全的地方。

---

## 第三步：配置环境变量

OpenClaw 的标准配置方式是在 `~/.openclaw/openclaw.json` 中统一填写 API Key，OpenClaw 运行时会自动将其注入给所有声明了该变量的技能，**无需在技能目录下放置任何 `.env` 文件**。

**文件路径**：`~/.openclaw/openclaw.json`

找到（或新建）`env` 字段，添加以下内容：

```json
{
  "env": {
    "GIGGLE_API_KEY": "your_giggle_api_key_here"
  }
}
```

将 `your_giggle_api_key_here` 替换为上一步获取的实际 API Key。

> **所有 giggle-* 技能共用此配置**，只需填写一次，无需为每个技能单独配置。

## 各技能使用说明（简明版）

### giggle-drama（AI 短视频）

根据你的故事创意，生成带分镜的短视频，支持多种画风。

**触发方式**（对 Agent 说）：

- "帮我生成一个关于...的视频"
- "我想做一个短视频，故事是..."
- "查看可用的视频风格"

**支持的画风**：

| 风格名称 | 特点 |
|---------|------|
| 3D古风 | 国风仙侠 CG 渲染质感 |
| 写实风格 | 电影级写实视觉 |
| 二次元 | 标准动漫画风 |
| 吉卜力 | 治愈系手绘插画 |
| 皮克斯 | 3D 卡通动画 |
| 2D漫剧 | 日漫国漫融合 |
| 国风水墨 | 传统水墨插画 |

**支持的视频比例**：16:9（横屏）、9:16（竖屏）

**时长选项**：30 秒 / 60 秒 / 120 秒（或自动）

**费用**：需要消耗 Giggle.pro 积分，生成前 Agent 会与你确认参数，生成过程中自动扣费。

**生成耗时**：通常 10-30 分钟，Agent 每 3 分钟自动汇报进度。

---

### giggle-aimv（AI 音乐视频 / MV）

根据音乐或描述，配合参考图片生成完整的 MV 视频。

**触发方式**（对 Agent 说）：

- "帮我生成一个 MV，音乐风格是..."
- "根据这段歌词生成音乐视频"
- "用我上传的音乐生成 MV"

**三种音乐模式**：

| 模式 | 适用场景 | 需要提供 |
|------|---------|---------|
| 提示词模式 | 描述想要的音乐风格 | 音乐风格描述 |
| 自定义模式 | 提供自己写的歌词 | 歌词 + 音乐风格 + 歌名 |
| 上传模式 | 已有音乐文件 | 平台上传的音乐资产 ID |

**参考图**：生成 MV 必须提供一张参考图，作为画面风格的基调。可自行提供，或让 Agent 使用 giggle-image 生成。

**支持的视频比例**：16:9（横屏）、9:16（竖屏）

**费用**：需要消耗 Giggle.pro 积分。

**生成耗时**：通常 3-10 分钟，Agent 每 3 分钟自动汇报进度。

---

### giggle-music（AI 音乐）

根据描述或歌词生成 AI 音乐。

**触发方式**（对 Agent 说）：

- "生成一首古风爱情歌曲"
- "帮我写一首歌，歌词是..."
- "生成一段轻松的背景音乐"

**支持的模式**：

| 模式 | 说明 |
|------|------|
| 描述模式 | 用文字描述风格，AI 自动创作（推荐） |
| 自定义模式 | 提供完整歌词，指定风格和歌名 |
| 纯音乐模式 | 生成无人声的背景音乐 |

**人声选项**：男声 / 女声 / 自动

**费用**：需要消耗 Giggle.pro 积分。

**生成耗时**：通常 1-3 分钟，完成后自动推送结果。

---

### giggle-image（AI 图像）

使用 Seedream 模型生成 AI 图像，支持文生图、图生图和多图融合。

**触发方式**（对 Agent 说）：

- "帮我画一张...的图片"
- "生成一张 16:9 的科幻城市壁纸"
- "把这张图片转成水彩画风格"
- "融合这几张图的风格生成新图"

**支持的图像比例**：

| 比例 | 适用场景 |
|------|---------|
| 16:9 | 横屏壁纸、文章封面（推荐） |
| 9:16 | 竖屏、手机壁纸、Stories |
| 1:1 | 社交媒体头像 |
| 3:4 / 4:3 | 通用竖版 / 横版 |
| 2:3 / 3:2 | 照片比例 |
| 21:9 | 超宽屏 |

**支持的生成模式**：

- **文生图**：纯文字描述生成图像
- **图生图**：提供参考图，生成相似风格或调整内容
- **多图融合**：最多 10 张参考图，融合风格生成新图

**费用**：需要消耗 Giggle.pro 积分。

**生成耗时**：通常 30-60 秒，完成后自动推送结果。

---

### giggle-screenplay（AI 剧本）

生成高质量中文剧本，支持多种风格。

**触发方式**（对 Agent 说）：

- "帮我写一个悬疑剧本"
- "生成一段古装宫廷的对白"
- "写一个姜文风格的短视频剧本"

**支持的创作风格**：

| 风格 | 特点 |
|------|------|
| 姜文风格（主推） | 黑色冲突、话里有话、高密度台词 |
| 甜宠爱情 | 温情治愈、误解靠近、轻冲突 |
| 悬疑惊悚 | 信息不对称、线索埋设、层层反转 |
| 古装宫廷 | 礼制权谋、古典韵味 |
| 喜剧 | 误会递进、性格极致化 |

**输出内容**：故事梗概 + 人物小传 + 分场大纲 + 分场剧本（含对白、动作、场面调度）

**费用**：无需积分，完全免费。

**无需 API Key**：可在未配置 `GIGGLE_API_KEY` 的环境下正常使用。

## 常见问题

### 提示"未设置 API 密钥"怎么办？

检查 `~/.openclaw/openclaw.json` 中是否正确填写了 `GIGGLE_API_KEY`。

```bash
# 查看当前配置
cat ~/.openclaw/openclaw.json
```

确认文件中存在以下内容（值已替换为实际 Key）：

```json
{
  "env": {
    "GIGGLE_API_KEY": "your_giggle_api_key_here"
  }
}
```

如果缺失，按照第三步重新配置。

### 积分不足怎么办？

Agent 会提示"积分不足，请充值后继续"。充值方式：

1. 登录 [giggle.pro](https://giggle.pro)
2. 进入个人中心，找到「积分充值」或「钱包」
3. 充值后告诉 Agent 继续，无需重新提交任务

### 视频/图片生成中途没有回复怎么办？

Agent 使用 Cron 机制自动轮询进度：

- **giggle-drama**：每 3 分钟自动汇报一次，共需 10-30 分钟
- **giggle-aimv**：每 3 分钟自动汇报一次，共需 3-10 分钟
- **giggle-music**：每 2 分钟自动检查一次，共需 1-3 分钟
- **giggle-image**：每 45 秒自动检查一次，共需 30-60 秒

如果等待超时仍未收到结果，可以询问 Agent："上次的视频/图片/音乐进展如何？"，Agent 会查询最近的任务状态。

### giggle-screenplay 可以不配置 API Key 使用吗？

可以。`giggle-screenplay` 是纯 LLM 技能，不调用 Giggle.pro API，无需配置 `GIGGLE_API_KEY`，也不消耗积分。
